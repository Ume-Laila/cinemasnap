# 🎬 CinemaSnap

> Discover your next favorite film — a cinematic movie discovery app




## 🛠️ Tech Stack

- React + Vite
- Tailwind CSS v4
- React Router DOM
- TMDB API (free)
- React Hot Toast
- React Icons

## ✨ Features

- 🔥 Browse trending, top rated, now playing movies
- 🔍 Search movies by title or genre  
- 🎭 Filter by mood — Happy, Scared, Romantic, Action, Emotional, Thoughtful
- 🔖 Save movies to personal watchlist (localStorage)
- 🎬 Full movie details with cast, trailers, similar movies
